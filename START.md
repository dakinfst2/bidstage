# 🚀 ЗАПУСК Encore

## Быстрый старт

### 1. Установите Python
Убедитесь, что у вас установлен Python 3.8 или выше:
```bash
python --version
```

### 2. Установите зависимости
```bash
pip install -r requirements.txt
```

### 3. Запустите сервер
```bash
python app.py
```

### 4. Откройте в браузере
```
http://127.0.0.1:5000
```

## ✅ Готово!

Вы увидите:
- 🎫 Главную страницу с hero-баннером
- 🎵 4 активных аукциона концертов
- ⏰ Таймеры обратного отсчета
- 🔐 Кнопки входа и регистрации

## 🧪 Первые шаги

1. **Зарегистрируйтесь**
   - Нажмите "Регистрация"
   - Введите данные
   - Войдите в систему

2. **Сделайте ставку**
   - Кликните на любой концерт
   - Введите сумму
   - Нажмите "Сделать ставку"

3. **Проверьте базу данных**
   - Откройте файл `bidstage.db`
   - Увидите свои данные!
