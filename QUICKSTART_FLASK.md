# ⚡ Быстрый запуск Encore Flask

## 🚀 Запуск за 3 шага

### Шаг 1: Установите зависимости
```bash
pip install -r requirements.txt
```

### Шаг 2: Запустите сервер
```bash
python app.py
```

### Шаг 3: Откройте в браузере
```
http://127.0.0.1:5000
```

**Готово!** База данных создастся автоматически с тестовыми данными.

---

## 📝 Что произойдет при первом запуске?

1. ✅ Создастся файл `bidstage.db` (SQLite база данных)
2. ✅ Создадутся 4 таблицы: User, Lot, Bid, Favorite
3. ✅ Добавятся 4 тестовых концерта
4. ✅ Сервер запустится на порту 5000
5. ✅ Откроется главная страница с аукционами

## 🧪 Как протестировать?

### 1. Регистрация
- Нажмите "Регистрация" в правом верхнем углу
- Введите имя пользователя, email и пароль
- Нажмите "Зарегистрироваться"

### 2. Просмотр лотов
- Прокрутите вниз до секции "Активные аукционы"
- Увидите 4 концерта с таймерами

### 3. Создание ставки
- Кликните на любую карточку концерта
- Введите сумму ставки (больше текущей цены + шаг)
- Нажмите "Сделать ставку"
- Ваша ставка сохранится в базе данных!

### 4. Избранное
- Нажмите на иконку сердца на карточке
- Лот добавится в избранное

## 🔧 Полезные команды

### Перезапустить с чистой БД
```bash
# Удалите базу данных
rm bidstage.db  # Linux/Mac
del bidstage.db  # Windows

# Запустите снова
python app.py
```

### Изменить порт
В файле `app.py` измените последнюю строку:
```python
app.run(debug=True, host='0.0.0.0', port=5001)  # Вместо 5000
```

### Отключить debug режим
```python
app.run(debug=False, host='0.0.0.0', port=5000)
```

## ❓ Частые проблемы

### Ошибка: "No module named 'flask'"
**Решение**: Установите зависимости
```bash
pip install -r requirements.txt
```

### Ошибка: "Address already in use"
**Решение**: Порт 5000 занят, измените на другой
```python
app.run(port=5001)
```

### Ошибка: "Unable to open database file"
**Решение**: Проверьте права доступа к папке
```bash
chmod 755 .  # Linux/Mac
```

### База данных не создается
**Решение**: Запустите Python напрямую
```bash
python
>>> from app import init_db
>>> init_db()
>>> exit()
```
