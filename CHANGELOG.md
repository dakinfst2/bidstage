# 📝 История изменений Encore

## Версия 2.0 — Flask Backend (14 мая 2026)

### ✨ Новые возможности

#### Backend
- ✅ **Flask сервер** с RESTful API
- ✅ **SQLite база данных** с 4 таблицами
- ✅ **SQLAlchemy ORM** для работы с БД
- ✅ **Регистрация и авторизация** с хешированием паролей
- ✅ **Сессии Flask** для хранения авторизации
- ✅ **11 API endpoints** для всех операций

#### База данных
- ✅ Таблица **User** (пользователи)
- ✅ Таблица **Lot** (лоты/концерты)
- ✅ Таблица **Bid** (ставки)
- ✅ Таблица **Favorite** (избранное)
- ✅ Автоматическая инициализация с тестовыми данными

#### Frontend
- ✅ **Интеграция с API** через fetch
- ✅ **Модальные окна** для входа и регистрации
- ✅ **Динамическое обновление** данных
- ✅ **Обработка ошибок** с понятными сообщениями

### 🔧 Технические улучшения

- Разделение на frontend и backend
- Структура проекта Flask (templates/, static/)
- requirements.txt для зависимостей
- .gitignore для игнорирования файлов
- Подробная документация в README.md

### 📂 Новая структура проекта

```
bidstage/
├── app.py                 # Flask приложение
├── requirements.txt       # Зависимости
├── templates/
│   └── index.html        # HTML шаблон
├── static/
│   └── script.js         # JavaScript с API
└── bidstage.db           # SQLite база (создается автоматически)
```

### 🚀 Запуск

```bash
pip install -r requirements.txt
python app.py
```

## Миграция с v1.0 на v2.0

### Что изменилось:
1. HTML переехал в `templates/index.html`
2. JavaScript переехал в `static/script.js`
3. Добавлен `app.py` с Flask сервером
4. Добавлена база данных SQLite

### Как обновиться:
```bash
# Установите зависимости
pip install -r requirements.txt

# Запустите новую версию
python app.py
```

### Обратная совместимость:
- Дизайн остался прежним
- Все функции работают как раньше
- Добавлены новые возможности
